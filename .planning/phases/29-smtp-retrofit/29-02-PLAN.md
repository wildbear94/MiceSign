**Fragment parameterization:**
- `th:fragment="layout (body)"` — parameter 이름은 반드시 `body` (page 측 `~{::body/*}` 과 매칭)
- `<div th:replace="${body}">` — parameter 변수를 DOM에 치환
- **중요 (Thymeleaf 3.1 semantics):** `<div th:replace="${body}">` 는 outer div 자체를 body parameter (`~{::body/*}`) 로 교체하므로 wrapper div 없이 inner body 내용만 layout 에 삽입됨. 결과 DOM 에는 "BODY PLACEHOLDER" 텍스트 또는 빈 wrapper `<div>` 가 남지 않음 — 각 이벤트 템플릿의 `<body>` 자식 요소들이 그대로 layout 의 `<td>` 셀 안으로 합성됨.
