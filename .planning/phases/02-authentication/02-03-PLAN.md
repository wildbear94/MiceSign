<!-- Backend API contracts from Plan 01 -->
POST /api/v1/auth/login: body { email, password, rememberMe } -> { success: true, data: { accessToken, user: { id, name, email, role, departmentId, mustChangePassword } } } + Set-Cookie refreshToken
POST /api/v1/auth/refresh: cookie refreshToken -> { success: true, data: { accessToken, user: {...} } } + Set-Cookie refreshToken
POST /api/v1/auth/logout: Bearer token + cookie -> { success: true, data: "..." } + clear cookie
Error responses: { success: false, error: { code: "AUTH_INVALID_CREDENTIALS"|"AUTH_ACCOUNT_LOCKED", message: "..." } }
Lockout response includes lockedUntil ISO timestamp and remainingAttempts integer
