
- 자격증명을 빈 default (`${DB_USER:}` / `${DB_PASS:}`) 로 복원 → env var 미설정 시 startup fail-fast (의도)
- `application-prod.yml.example` 신설 — 운영자가 `/etc/micesign/.env.production` 작성 시 참조할 env 변수 카탈로그
- `.gitignore` 의 `.env*` 차단 패턴 검증 (이미 차단되어 있다면 변경 없음)

systemd unit 파일 작성 + EnvironmentFile 적용 + `chmod 600` 등 **운영 시점 절차** 는 본 plan 이 아닌 33-02 (SMTP-RUNBOOK) 에서 단계별로 문서화한다. 본 plan 은 코드/설정 위생만.

Output: hardcoded secret 이 제거된 application-prod.yml + 운영자가 참조할 example 파일.
