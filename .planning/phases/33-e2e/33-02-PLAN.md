    ## §6. 트러블슈팅 + 주기적 점검

    ### 6.1 BaseUrlGuard 가 startup 실패시킴
    증상: `journalctl -u micesign` 에 `IllegalStateException: app.base-url contains 'localhost'`.
    원인: `/etc/micesign/.env.production` 의 `APP_BASE_URL` 이 미주입 또는 localhost 포함.
    조치:
    1. `sudo cat /etc/micesign/.env.production | grep APP_BASE_URL` 확인.

    ### 6.2 SMTP 발송이 PENDING 에서 진행 안 함

        error_message = '[수동전환] 10분 이상 PENDING — 서버 재시작 또는 SMTP 장애 의심',

    ### 6.3 주간 점검 (Phase 29 D-B6 인계 — push 알림 deferred 의 보완)

    매주 1회 (월요일 권장) 다음 SQL 실행:

    \`\`\`sql
    -- 지난 7일 발송 실패 건수
    SELECT event_type, COUNT(*) AS failed_cnt
    FROM notification_log
    WHERE status = 'FAILED'
      AND created_at >= DATE_SUB(NOW(), INTERVAL 7 DAY)
    GROUP BY event_type
    ORDER BY failed_cnt DESC;
    \`\`\`

    1. `error_message` 컬럼으로 원인 구분 (SMTP timeout / auth fail / template render error).
