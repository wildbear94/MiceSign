1. Create `frontend/src/features/document/utils/evaluateConditions.ts`:
   - Export `FieldVisibility` interface: `{ visible: boolean; conditionallyRequired: boolean }`
   - Export `evaluateConditions(rules: ConditionalRule[], formValues: Record<string, unknown>, fields: { id: string; required: boolean }[]): Map<string, FieldVisibility>`
   - Initialize all fields as `{ visible: true, conditionallyRequired: false }`
   - For each rule, evaluate conditions using `evaluateMatch` helper
   - `evaluateMatch`: for each condition, compare `formValues[sourceFieldId]` against operator:
     - `equals`: `String(val ?? '') === cond.value`
     - `not_equals`: `String(val ?? '') !== cond.value`
     - `is_empty`: `val == null || String(val) === ''`
     - `is_not_empty`: `val != null && String(val) !== ''`
   - matchType 'all': `results.every(Boolean)`, 'any': `results.some(Boolean)`
   - `show` action: if NOT matched, set visible=false
   - `hide` action: if matched, set visible=false
   - `require` action: if matched, set conditionallyRequired=true
   - Per D-07: existing required=true stays required regardless; conditionallyRequired is additive

2. Create `frontend/src/features/document/utils/executeCalculations.ts`:
   - Export `executeCalculation(rule: CalculationRule, formValues: Record<string, unknown>): number`
   - For each sourceField in rule.sourceFields:
     - If contains `.`: split on first `.` to get `[tableId, colId]`, extract `formValues[tableId]` as array, map each row's `colId` to Number or 0
     - Else: `[Number(formValues[fieldId]) || 0]`
   - Flatten all values, then:
     - SUM/ADD: `values.reduce((a, b) => a + b, 0)`
     - MULTIPLY: `values.reduce((a, b) => a * b, 1)`
     - COUNT: `values.length`
   - Round result to 2 decimal places: `Math.round(result * 100) / 100`

3. Create `frontend/src/features/document/utils/detectCircularDeps.ts`:
   - Export `detectCircularDeps(conditionalRules: ConditionalRule[], calculationRules: CalculationRule[]): string[] | null`
   - Build directed graph: Map<string, Set<string>>
     - For each conditional rule: source -> target edges (source = each condition.sourceFieldId, target = rule.targetFieldId)
     - For each calculation rule: source -> target edges (source = each sourceField, target = rule.targetFieldId)
   - DFS 3-color (WHITE=0, GRAY=1, BLACK=2):
     - When visiting node already GRAY: cycle found — reconstruct path
     - Return cycle path as string array (field IDs in cycle order)
     - Return null if no cycle

   - `__tests__/executeCalculations.test.ts`: test SUM/ADD/MULTIPLY/COUNT with flat fields, table.column references, non-numeric handling, empty values
   - `__tests__/detectCircularDeps.test.ts`: test empty rules, linear chain (no cycle), simple A<->B cycle, 3-node cycle, mixed conditional+calculation cycle, self-reference
  </action>
