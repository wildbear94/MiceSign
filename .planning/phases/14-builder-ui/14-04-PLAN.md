2. Create `PropertyBasicTab.tsx`:
   - Props: `{ field: FieldDefinition; onUpdateField: (fieldId: string, changes: Partial<FieldDefinition>) => void; onUpdateFieldConfig: (fieldId: string, config: Partial<FieldConfig>) => void; }`
   - **Use controlled inputs (NOT react-hook-form).** Rationale: The property panel edits a field object managed by useBuilderReducer. Using react-hook-form here would cause an infinite loop — the form's defaultValues would need to reset every time the selected field changes or a different field is clicked, triggering re-renders that conflict with RHF's internal state tracking (RESEARCH Pitfall 2). Controlled inputs with direct onChange -> dispatch are simpler and avoid this issue since the source of truth is the reducer state, not form state.
   - Use onChange handlers that dispatch immediately:
   - Fields:
     - Label input: `<input value={field.label} onChange={e => onUpdateField(field.id, { label: e.target.value })} className="w-full px-3 py-2 border border-gray-300 rounded-lg text-sm focus:ring-2 focus:ring-blue-600 focus:border-transparent" />`
     - Required toggle: `<label className="flex items-center gap-2"><input type="checkbox" checked={field.required} onChange={e => onUpdateField(field.id, { required: e.target.checked })} className="rounded border-gray-300 text-blue-600 focus:ring-blue-600" /><span className="text-sm">{t('templates.fieldRequired')}</span></label>`
     - Placeholder input (show for text, textarea, number only): `<input value={field.config?.placeholder ?? ''} onChange={e => onUpdateFieldConfig(field.id, { placeholder: e.target.value })} ... />`
     - Description input (optional field help text): text input for field.config description
     - Static content textarea (show for staticText only): `<textarea value={field.config?.content ?? ''} onChange={e => onUpdateFieldConfig(field.id, { content: e.target.value })} ... />`
     - Default value (show for hidden only): `<input value={field.config?.defaultValue ?? ''} onChange={e => onUpdateFieldConfig(field.id, { defaultValue: e.target.value })} ... />`
   - Conditionally hide irrelevant fields based on `field.type`:
     - text/textarea: show label, required, placeholder
     - number: show label, required, placeholder
     - date: show label, required
     - select: show label, required (options editor separate)
     - table: show label (columns editor separate)
     - staticText: show content textarea only
     - hidden: show label, defaultValue
   - Each field row: `<div className="mb-4"><label className="block text-sm font-semibold text-gray-700 dark:text-gray-300 mb-1">{label}</label>{input}</div>`
