<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 미리보기 입력 -> formValues state | 사용자 입력이 React state로 저장 (서버 전송 안 함) |
| conditionalRules -> schemaDefinition JSON | 프론트엔드 state가 백엔드 저장 payload에 포함 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-24-04 | Tampering | schemaDefinition JSON 저장 | accept | 백엔드는 JSON을 그대로 저장 (D-26). 관리자(ADMIN/SUPER_ADMIN)만 양식 편집 가능하므로 권한 보호됨. |
| T-24-05 | Information Disclosure | 미리보기 formValues | accept | formValues는 React 컴포넌트 로컬 state. 서버에 전송되지 않으며 모달 닫기 시 폐기. |
| T-24-06 | Denial of Service | 대량 필드 시 evaluateConditions 성능 | accept | 필드당 규칙 1개 (D-02), 50인 회사에서 필드 수 < 100개 예상. evaluateConditions는 O(N) 순회로 성능 문제 없음. |
</threat_model>
