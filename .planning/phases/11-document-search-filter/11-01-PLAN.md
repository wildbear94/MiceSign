6. **Create `DocumentRepositoryCustomImpl.java`** implementing the interface:

- **Tab scope logic (per D-07, D-08, D-09, D-10):**
  - `MY` -> `where.and(doc.drafter.id.eq(userId))`
  - `APPROVAL` -> subquery: `where.and(doc.id.in(JPAExpressions.select(approvalLine.document.id).from(approvalLine).where(approvalLine.approver.id.eq(userId))))`
  - `ALL` -> no additional predicate (RBAC enforced at controller)
- **Keyword search (per D-05, D-06):** If keyword is not blank, escape LIKE wildcards (`%`, `_`, `\`) then wrap with `%keyword%`:
```java
if (condition.keyword() != null && !condition.keyword().isBlank()) {
    String escaped = escapeLikePattern(condition.keyword().trim());
    String kw = "%" + escaped + "%";
    where.and(
        doc.title.likeIgnoreCase(kw)
            .or(doc.docNumber.likeIgnoreCase(kw))
            .or(doc.drafter.name.likeIgnoreCase(kw))
    );
}
```
- **Filters (per D-11, D-12):** Each filter is AND-combined, only when non-null:
  - `status != null` -> `where.and(doc.status.eq(condition.status()))`
  - `templateCode != null` -> `where.and(doc.templateCode.eq(condition.templateCode()))`
  - `startDate != null` -> `where.and(doc.createdAt.goe(condition.startDate().atStartOfDay()))`
  - `endDate != null` -> `where.and(doc.createdAt.lt(condition.endDate().plusDays(1).atStartOfDay()))`
- **Query execution (per D-17):** Sort by `doc.createdAt.desc()`, use offset/limit from pageable. Use `leftJoin(doc.drafter).fetchJoin()` and `leftJoin(doc.drafter.department).fetchJoin()` to avoid N+1.
- **Count query:** Separate `queryFactory.select(doc.count()).from(doc).where(where).fetchOne()` for pagination total.
- Return `new PageImpl<>(content, pageable, total)`.
- **Add private helper:**
```java
private String escapeLikePattern(String input) {
    return input.replace("\\", "\\\\").replace("%", "\\%").replace("_", "\\_");
}
```

```java
@Transactional(readOnly = true)
public Page<DocumentResponse> searchDocuments(DocumentSearchCondition condition, Long userId, Pageable pageable) {
    Page<Document> documents = documentRepository.searchDocuments(condition, userId, pageable);
    
    Map<String, String> templateNameMap = approvalTemplateRepository.findByIsActiveTrueOrderBySortOrder()
            .stream()
            .collect(Collectors.toMap(ApprovalTemplate::getCode, ApprovalTemplate::getName));
    
    return documents.map(doc -> documentMapper.toResponse(doc,
            templateNameMap.getOrDefault(doc.getTemplateCode(), doc.getTemplateCode())));
}
```

2. **Add search endpoint to `DocumentController.java`** (per RESEARCH.md API contract):
```java
@GetMapping("/search")
public ApiResponse<Page<DocumentResponse>> searchDocuments(
        @AuthenticationPrincipal CustomUserDetails user,
        @RequestParam(defaultValue = "MY") DocumentSearchCondition.SearchTab tab,
        @RequestParam(required = false) String keyword,
        @RequestParam(required = false) DocumentStatus status,
        @RequestParam(required = false) String templateCode,
        @RequestParam(required = false) @DateTimeFormat(iso = DateTimeFormat.ISO.DATE) LocalDate startDate,
        @RequestParam(required = false) @DateTimeFormat(iso = DateTimeFormat.ISO.DATE) LocalDate endDate,
        @RequestParam(defaultValue = "0") int page,
        @RequestParam(defaultValue = "20") int size) {
    
    // RBAC enforcement for ALL tab (per D-09, Pitfall 4)
    if (tab == DocumentSearchCondition.SearchTab.ALL) {
        String role = user.getRole();
        if (!"SUPER_ADMIN".equals(role) && !"ADMIN".equals(role)) {
            throw new BusinessException("AUTH_FORBIDDEN", "권한이 없습니다.", 403);
        }
    }
    
    DocumentSearchCondition condition = new DocumentSearchCondition(
            tab, keyword, status, templateCode, startDate, endDate);
    PageRequest pageable = PageRequest.of(page, size);
    
    return ApiResponse.ok(documentService.searchDocuments(condition, user.getUserId(), pageable));
}
```
Import: `org.springframework.format.annotation.DateTimeFormat`, `java.time.LocalDate`, `com.micesign.dto.document.DocumentSearchCondition`.
