<threat_model>
- **XSS via schema labels**: Field labels from JSON schema rendered in UI. Mitigation: React JSX auto-escapes all text content; no `dangerouslySetInnerHTML`.
- **staticText content injection**: staticText field's `content` rendered as display text. Mitigation: Render as text node, not HTML.
- **Prototype pollution via formData**: JSON.parse of formData could inject `__proto__`. Mitigation: formData is always a flat key-value object matching schema field IDs; no deep merge operations.
</threat_model>
