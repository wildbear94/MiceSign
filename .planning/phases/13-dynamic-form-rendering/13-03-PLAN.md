196행 근처의 "Unknown template" 분기를 수정:

기존:
```typescript
const templateEntry = TEMPLATE_REGISTRY[resolvedTemplateCode];

  return <div>Unknown template: {resolvedTemplateCode}</div>;

```

변경:
```typescript

```

"Unknown template" 에러 분기 제거 -- isDynamic이면 DynamicForm 렌더링.

EditComponent 렌더링 부분을 조건부로 변경:
```typescript
{isDynamic ? (
  <DynamicForm
    documentId={savedDocId}
    templateCode={resolvedTemplateCode}
    initialData={existingDoc ? {
      title: existingDoc.title,
      bodyHtml: existingDoc.bodyHtml ?? undefined,
      formData: existingDoc.formData ?? undefined,
    } : undefined}
    onSave={handleSave}
    schemaDefinition={existingDoc?.schemaDefinitionSnapshot ? JSON.parse(existingDoc.schemaDefinitionSnapshot) as SchemaDefinition : undefined}
  />
) : EditComponent ? (
  <EditComponent
    documentId={savedDocId}
    initialData={existingDoc ? {
      title: existingDoc.title,
      bodyHtml: existingDoc.bodyHtml ?? undefined,
      formData: existingDoc.formData ?? undefined,
    } : undefined}
    onSave={handleSave}
  />
) : null}
```

DynamicForm은 자체 form 내에서 모든 onSave 로직을 처리하므로, DocumentEditorPage 수정은 렌더링 분기 변경만 필요 (per D-02).
