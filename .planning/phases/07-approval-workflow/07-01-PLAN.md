## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> API | Untrusted approval line data crosses to backend (approverId, lineType, stepOrder) |
| approver -> approval action | User claims to be the current approver to approve/reject |
| drafter -> withdraw | User claims to be the drafter and that no approver has acted |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-05 | Information Disclosure | DocumentService.getDocument | mitigate | RBAC check: drafter + approval line users + admin only [VERIFIED existing] |
| T-07-06 | Repudiation | ApprovalService | mitigate | AuditLogService records all approval actions [VERIFIED existing] |
