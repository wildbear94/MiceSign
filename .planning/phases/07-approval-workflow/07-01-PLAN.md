**`approve(Long userId, Long lineId, String comment)` method:**
1. Load approval line with pessimistic lock: `approvalLineRepository.findByIdForUpdate(lineId).orElseThrow(() -> new BusinessException("APR_NOT_FOUND", "결재선을 찾을 수 없습니다.", 404))`
2. Load document via `line.getDocument()`
3. Validate document status is SUBMITTED: throw `BusinessException("APR_DOC_NOT_SUBMITTED", "제출 상태의 문서만 결재할 수 있습니다.", 400)` if not
4. Validate line belongs to this user: `line.getApprover().getId().equals(userId)`, throw `BusinessException("APR_NOT_AUTHORIZED", "본인의 결재만 처리할 수 있습니다.", 403)` if not
5. Validate line status is PENDING: throw `BusinessException("APR_ALREADY_PROCESSED", "이미 처리된 결재입니다.", 400)` if not
6. Validate it's this user's turn: `line.getStepOrder().equals(document.getCurrentStep())`, throw `BusinessException("APR_NOT_YOUR_TURN", "현재 결재 순서가 아닙니다.", 400)` if not
7. Update line: `status = APPROVED`, `comment = comment`, `actedAt = LocalDateTime.now()`
8. Check if all lines at current step are done (query `findByDocumentIdAndStepOrder` — in this MVP, 1 approver per step, so this completes the step)
9. Find next step: query for next PENDING line with stepOrder > currentStep where lineType is APPROVE or AGREE, ordered by stepOrder ASC. If found, `document.setCurrentStep(nextLine.getStepOrder())`. If not found, `document.setStatus(DocumentStatus.APPROVED)`, `document.setCompletedAt(LocalDateTime.now())` (per APR-05).
10. Save document.

**`reject(Long userId, Long lineId, String comment)` method:**
1. Same validation steps 1-6 as approve
2. Validate comment is not blank (per D-13): throw `BusinessException("APR_COMMENT_REQUIRED", "반려 사유를 입력해주세요.", 400)` if comment is null or blank
3. Update line: `status = REJECTED`, `comment = comment`, `actedAt = LocalDateTime.now()`
4. Immediately set `document.setStatus(DocumentStatus.REJECTED)`, `document.setCompletedAt(LocalDateTime.now())` (per APR-04)
5. Note: Per FSD, remaining PENDING lines stay PENDING — do NOT change them to SKIPPED on rejection
6. Save document.

**`getPendingApprovals(Long userId, Pageable pageable)` method:**
1. Call `approvalLineRepository.findPendingByApproverId(userId, pageable)`
2. Map each ApprovalLine to PendingApprovalResponse using document and drafter info
3. Build template name lookup from ApprovalTemplateRepository
4. Return `Page<PendingApprovalResponse>`

**`getCompletedDocuments(Long userId, Pageable pageable)` method:**
1. Query documents where `drafterId = userId` AND `status = APPROVED`
2. Use `documentRepository.findByDrafterIdAndStatusIn(userId, List.of(DocumentStatus.APPROVED), pageable)`
3. Map to DocumentResponse using DocumentMapper
4. Return page

**2. Create ApprovalController.java:**

```java
@RestController
@RequestMapping("/api/v1/approvals")
public class ApprovalController {
    // Inject: ApprovalService

    @PostMapping("/{lineId}/approve")
    public ApiResponse<Void> approve(
            @AuthenticationPrincipal CustomUserDetails user,
            @PathVariable Long lineId,
            @RequestBody(required = false) ApprovalActionRequest request) {
        approvalService.approve(user.getUserId(), lineId, request != null ? request.comment() : null);
        return ApiResponse.ok(null);
    }

    @PostMapping("/{lineId}/reject")
    public ApiResponse<Void> reject(
            @AuthenticationPrincipal CustomUserDetails user,
            @PathVariable Long lineId,
            @Valid @RequestBody ApprovalActionRequest request) {
        approvalService.reject(user.getUserId(), lineId, request.comment());
        return ApiResponse.ok(null);
    }

    @GetMapping("/pending")
    public ApiResponse<Page<PendingApprovalResponse>> getPendingApprovals(
            @AuthenticationPrincipal CustomUserDetails user,
            @RequestParam(defaultValue = "0") int page,
            @RequestParam(defaultValue = "20") int size) {
        PageRequest pageable = PageRequest.of(page, size);
        return ApiResponse.ok(approvalService.getPendingApprovals(user.getUserId(), pageable));
    }

    @GetMapping("/completed")
    public ApiResponse<Page<DocumentResponse>> getCompletedDocuments(
```
