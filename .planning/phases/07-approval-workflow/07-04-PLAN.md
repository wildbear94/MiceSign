**1. Create `ApprovalActionBar.tsx`:**

Props: `document: DocumentDetailResponse`, `currentUserId: number`

Implementation per UI-SPEC:

Determine visibility conditions using `document.currentStep` directly from the backend response (Plan 01 added `Integer currentStep` to `DocumentDetailResponse`, Plan 03 added `currentStep: number | null` to the frontend type):
- `isCurrentApprover`: document.status === 'SUBMITTED' AND there exists an approval line where `approver.id === currentUserId` AND `status === 'PENDING'` AND `stepOrder === document.currentStep`
- `isDrafter`: document.drafter.id === currentUserId
- `canWithdraw`: isDrafter AND document.status === 'SUBMITTED' AND all approval lines at current step (`stepOrder === document.currentStep`) are PENDING (per D-21)
- `canResubmit`: isDrafter AND (document.status === 'REJECTED' OR document.status === 'WITHDRAWN') (per D-27)

Get the current user's approval line ID for approve/reject actions: find the line where `approver.id === currentUserId` and `status === 'PENDING'`.

State: `dialogMode: 'approve' | 'reject' | null`, `showWithdrawConfirm: boolean`, `showResubmitConfirm: boolean`

Use hooks: `useApprove()`, `useReject()`, `useWithdraw()`, `useRewrite()` from the approval/document hooks.

Render (per UI-SPEC positioning and styling):
```
<div className="flex items-center gap-3 mb-6">
  {isCurrentApprover && (
    <>
      <button onClick={() => setDialogMode('approve')}
        className="bg-blue-600 hover:bg-blue-700 text-white h-11 px-5 rounded-lg text-sm font-medium">
        결재하기
      </button>
      <button onClick={() => setDialogMode('reject')}
        className="bg-white border border-red-300 text-red-600 hover:bg-red-50 h-11 px-5 rounded-lg text-sm font-medium">
        반려하기
      </button>
    </>
  )}
  <div className="flex-1" />
  {canWithdraw && (
    <button onClick={() => setShowWithdrawConfirm(true)}
      className="bg-white border border-gray-300 text-gray-700 hover:bg-gray-50 h-11 px-5 rounded-lg text-sm font-medium">
      문서 회수
    </button>
  )}
  {canResubmit && (
    <button onClick={() => setShowResubmitConfirm(true)}
      className="bg-blue-600 hover:bg-blue-700 text-white h-11 px-5 rounded-lg text-sm font-medium">
      재기안하기
    </button>
  )}
</div>
```
