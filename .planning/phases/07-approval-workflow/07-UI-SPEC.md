## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, badge internal padding |
| sm | 8px | Compact element spacing, timeline connector gaps |
| md | 16px | Default element spacing, modal/panel internal padding |
| lg | 24px | Section padding, modal body padding (`p-6`) |
| xl | 32px | Gap between approval line editor and form content |
| 2xl | 48px | Not used this phase |
| 3xl | 64px | Not used this phase |

Exceptions:

| Value | Justification |
|-------|---------------|
| 44px (`h-11`) | Touch target minimum per WCAG 2.5.8. Used for approve/reject/withdraw/resubmit buttons. Matches existing Phase 4-6 button height. |

Source: Identical to Phase 4/5/6 spacing scale.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `white` / `gray-900` (dark) | Page background, main content area, modal background |
| Secondary (30%) | `gray-50` / `gray-800` (dark) | Timeline background, org tree picker panel, section headers |
| Accent (10%) | `blue-600` | Approve button, active nav item, primary CTA |
| Destructive | `red-600` | Reject button, rejection badge, withdraw confirmation |

Accent reserved for:
- Approve button (`bg-blue-600 hover:bg-blue-700 text-white`) -- primary approval action
- "Í≤įžě¨žěź ž∂ĒÍįÄ" button in approval line editor (`bg-blue-600`)
- Active timeline step indicator

| State | Color | Usage |
|-------|-------|-------|

| Rejected (badge) | `red-100` / `red-700` | Approval line step badge for rejections |
