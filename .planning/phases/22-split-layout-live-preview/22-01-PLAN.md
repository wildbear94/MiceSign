**PreviewFieldRenderer.tsx** — per D-04, lightweight read-only field renderer:
- Props: `{ field: SchemaField }` imported from `../SchemaFieldEditor/types`
- Use `useTranslation('admin')` for i18n
- Switch on `field.type` to render disabled inputs for each of the 8 SchemaFieldType values:
  - `text`: `<input type="text" disabled>` with `field.config.placeholder`
  - `textarea`: `<textarea disabled rows={3}>` with `field.config.placeholder`
  - `number`: `<input type="number" disabled>` with unit suffix `field.config.unit` shown as `<span>` beside input
  - `date`: `<input type="date" disabled>`
  - `select`: `<select disabled>` with `<option>` for each `field.config.options` item, default option text `t('templates.selectPlaceholder')`
  - `staticText`: `<p>` showing `field.config.content` or `t('templates.staticTextPlaceholder')` fallback
  - `hidden`: return `null` (hidden fields not rendered in preview)
  - `table`: placeholder `<div>` with `t('templates.tableFieldPlaceholder')` text
  - `default`: return `null`
- Each field wrapped in a `<div>` with:
  - `<label>` showing `field.label || t('templates.noLabel')` with `text-sm font-medium text-gray-700 dark:text-gray-300 mb-1`
  - Required indicator: `{field.required && <span className="text-red-500 ml-1">*</span>}`
- Disabled input classes (per UI-SPEC): `bg-gray-50 dark:bg-gray-700 text-gray-400 border-gray-300 dark:border-gray-600 rounded-lg text-sm`
- Input height: `h-10 px-3` for single-line inputs

**FormPreview.tsx** — per D-04, D-05:
- Props: `{ fields: SchemaField[]; templateName?: string }`
- Use `useTranslation('admin')`
- If `fields.length === 0`: render empty state with `t('templates.previewEmpty')` centered, `text-gray-400 dark:text-gray-500`
- If fields exist: render paper document card per D-05:
  - Outer: `bg-white dark:bg-gray-800 rounded-lg shadow-md p-8 max-w-2xl mx-auto`
  - Template name header: `{templateName && <h3 className="text-lg font-semibold text-center mb-6 border-b border-gray-200 dark:border-gray-700 pb-4">{templateName}</h3>}`
  - Fields container: `<div className="space-y-4">` mapping over `fields.filter(f => f.type !== 'hidden')` with `<PreviewFieldRenderer key={field.id} field={field} />`
